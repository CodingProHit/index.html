How to find the area of a triangle

Example Problem

Let's say that B was equal to 43 inches and C was equal to 78 inches. What is the area of the triangle.

Solution

To find the area of this triangle. You have to do 1/2 times Width Times Height. So you have to multiply the height and the Width and then divide the answer by 2. So I did 43 X 78 and got 3354 inches. Then I divided that by 1/2 and got 1677 inches. So now you know that the area is 1677 inches SQUARED

Solution

Dividing Fractions By Whole numbers.

Example Problem/h2>

3/4 divided by 7

Solution

For this type of problem. We use the KCF method. KCF stands for Keep Change Flip. You keep the fraction the way it is. Then you change the whole number by putting it over one. So it is still the same number, just in fraction form. So in this case it is 7/1. So now we have 3/4 divided by 7/1. Then we flip the fraction that used to be a whole number. So now we have 1/7. But now instead of dividing you would multiply. So 3/4 * 1/7. So now you would get 3/28

Finding Common Denominators In Fractions

Example Problem/h2>

What is the common denominator between 3/4 and 5/6

Solution

For this problem, we would have to find the least common multiple between the two denominators. The two denominators are 4 and 6. So we could make lists of the multiples of 4 and 6. For 4 it is 4,8,12,16 etc. For 6 it is 6,12, and here! We do not have to do more because we see the same number on both lists, this number is 12. So the common denominator is 12. If we wanted to find a fraction equevilrnt to 3/4, we know that 4 * 3 is 12, so we have to multiply the numberator by 3 too so we do 3 * 3 to get 9. So 3/4 is equal to 9/12!
